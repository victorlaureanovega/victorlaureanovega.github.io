¿Qué es HTML? (Explicar las etiquetas head y body)

HTML es un lenguaje de programación que se utiliza para estructurar páginas web. La etiqueta 'head' almacena la información que no se muestra en el navegador, pero que es muy importante que este conozca. En ella puede dársele un título a la página, o enlazar el HTML con un CSS. En la etiqueta body es donde se encuentran los elementos principales de la página que el usuario podrá ver, tales como imágenes, texto, enlace a otras páginas, videos, etc.

Explicar las etiquetas section, article, aside, header y footer

La etiqueta header representa el encabezado de la página o de una sección (con algún logo, menú, título, etc.), mientras que footer contiene la información al pie de la página o de una sección (referencias adicionales). section sirve para agrupar elementos que tienen similitudes temáticas, article es para contenido independiente como publicaciones o comentarios, y aside es para información adicional, como la de las barras laterales.

CSS

¿Qué es CSS y cuál es su propósito?

CSS es un lenguaje de programación utilizado para personalizar los elementos de un documento escrito en un lenguaje de marcado, como un HTML o un XML, es decir, para hacer el contenido legible, resaltar algunas cosas por encima de otras y contribuir estéticamente al diseño.

Tres formas de incluir estilos en un HTML

link rel="stylesheet" href="archivo.css" en el header

Etiqueta style también en el header

En la misma línea: p style="color: blue; font-size: 18px;">Texto con estilo en línea /p

¿Qué es un selector? ¿Qué es una Regla CSS?

Un selector es una palabra que hace referencia a una etiqueta o a un elemento del HTML, y sirve para indicar al CSS qué se aplicará a qué elemento. Una Regla CSS es el conjunto de líneas de código que aplican a un selector.

Explicar Tag Selector, Class Selector, Tag and Class Selector y Id Selector

Tag sirve para seleccionar todos los elementos de un tipo específico, y hace referencia a etiquetas HTML (p {}, h1 {}, h3 {}). Class es para clases creadas por el usuario, y se define con un punto antes del nombre (.clase {}, por ejemplo). Tag and Class aplica a tipos específicos de una clase, y se denota: p.clase {}. Id se utiliza en elementos a los se atribuyó determinado identificador en el HTML: #id {}.

Explicar el modelo de caja de CSS con un ejemplo

Este modelo representa la forma en que se estructura el contenido de los elementos contenidos en el body de un HTML. Nos ayuda a comprender cuánto espacio ocupa cada elemento, y propone cuatro partes: content (el contenido en sí, ya sea un texto o una imagen), padding (el espacio entre el contenido y el borde de la caja), border (contorno de la caja) y margin espacio entre la caja y otros elementos.

¿Cómo se puede cambiar el color de la letra con CSS?

Se puede escribir una instrucción parecida a la siguiente, ya sea utilizando colores "naturales" (es decir, escribiendo el nombre, i.e.: red, blue, violet, pink), utilizando valores hexadecimales, RGB, RGBA o HSL: p { color: #45e; }
